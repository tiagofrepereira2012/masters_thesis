\chapter{Conclusões Parciais}
\label{cap:conclusion}

O estudo de detecção de ataques de \textit{spoofing} em sistemas de autenticação de face é bastante recente. Diversas contramedidas vem sendo publicadas nos últimos meses, porém cada trabalho apresenta avaliações utilizando métricas diferentes em bases de dados muitas vezes privadas impossibilitando a reprodução de resultados e eventual comparação. 

A principal contribuição esperada para este projeto de mestrado é prover uma metodologia de comparação justa entre contramedidas. Contramedidas serão comparadas seguindo o seguinte dois protocolos medindo a performance na detecção dos ataques em: 
\begin{itemize}
        \item Bases de dados de referência;
        \item Um cenário similar a um cenário real.
\end{itemize}
